Arx Libertatis is a cross-platform port of Arx Fatalis, a first-person role-playing game.

We have not yet released an stable version, but the game is already playable under Windows and Linux. Just grab the source code on GitHub and follow the provided instructions. The wiki also has operating system-specific build instructions.

While the source code is available under the GPL 3+ license, you need to obtain the game data from the original Arx Fatalis to play Arx Libertatis.

The best way to contact the developers is through IRC in #arxfatalis on freenode. You can also find us on reddit.
